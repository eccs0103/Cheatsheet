# Cheatset (Шпаргалка)
Программа для решения тестов.  
© Adaptive Core 2022  
- - -
## Руководство
Для работы программы требуется база данных. База данных должен быть файл JSON со структурой:  
```lang-js
{
	title: String,
	date: Number,
	poles: Array<{
		question: String,
		answer: Number,
		cases: Array<String>,
	}>,
}
```
и может быть импортирован с помощью ссылки, через кнопку Import.
