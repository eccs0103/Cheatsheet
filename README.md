# Cheatset (Шпаргалка)
Программа для решения тестов.  
© Adaptive Core 2023  
- - -
## Руководство
Для работы программы требуется лист. Лист должен быть файл JSON со структурой:  
```lang-js
{
	title: String,
	poles: Array<{
		question: String,
		answer: Number,
		cases: String | Array<String>,
	}>,
}
```
и может быть загружен с устройства или импортирован с помощью ссылки.
